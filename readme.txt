== Description ==

LIQUID CONNECT is an engagement tool for web sites. Easily manage with pop-up banner, random banner, target setting, trigger setting, etc. with widget.

= Contents =

* Images, Button, Link
* Button Color
* Custom HTML, Shortcode
* A/B Test

= Target =

* New Visitor, Returning Visitor
* Campaign Source (Regex)
* Desktop, Mobile

= Trigger =

* Page Slug (Regex)
* Scroll Position
* Period
* Banner, Modal (left, center, right)

= Other features =

* View & Click Report (Google Analytics Event: gtag.js)
* Responsive Web Design
* You can enable / disable on the administration screen.
* You can also check the latest information on [LIQUID PRESS](https://lqd.jp/wp/).

== Installation ==

1. Upload 'liquid-connect' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set 'LIQUID CONNECT' in the 'Widget' menu
